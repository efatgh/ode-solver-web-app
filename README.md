# Differential Equation Solver

## Description
A web-based tool for solving ordinary differential equations (ODEs) using Python, SymPy, and Gradio.  
The app provides an interactive interface for entering differential equations and viewing the solutions in a readable mathematical format.

## Live Demo
[Open the app on Hugging Face Spaces]([https://huggingface.co/spaces/Efat98/differential-equation-tool])
Sample Inputs to Try:
Linear First Order: Derivative(y(x), x) + y(x) - x = 0
Separable: Derivative(y(x), x) - y(x)**2 * x = 0
Second Order: Derivative(y(x), x, 2) + y(x) = 0
